# Crypto Volatility and Risk Analyzer

## About the Project
This project is a part of the **Infosys Springboard Virtual Internship 6.0**.

The aim of the project is to analyze the volatility and risk levels of various cryptocurrencies using data analytics and machine learning techniques.

## Objectives
- Analyze historical crypto price data
- Measure volatility and risk levels
- Identify high-risk and low-risk cryptocurrencies
- Visualize price trends and market fluctuations
- Assist users in smarter investment decisions
